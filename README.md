# SEO Audit Webform App

Frontend application for submitting SEO audit requests to n8n.

## Features
- Simple intake form
- Sends requests to n8n webhook
- Loading state while audit generates
- Redirects user to generated GitHub Pages audit

- index.html
- style.css
- script.js
